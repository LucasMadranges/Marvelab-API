Marvelab API Documentation

Welcome to the Marvelab API documentation. This API provides access to research data including notes, interpretations, resources, methodologies, and experiments.

Base URL

All API endpoints are relative to: http://localhost:3000/api

Endpoints

Notes

GET/notes

Returns all notes.

GET/notes/:id

Returns a specific note by ID.

Interpretations

GET/interpretations

Returns all interpretations.

GET/interpretations/:id

Returns a specific interpretation by ID.

Resources

GET/resources

Returns all resources.

GET/resources/:id

Returns a specific resource by ID.

Methodologies

GET/methodologies

Returns all methodologies.

GET/methodologies/:id

Returns a specific methodology by ID.

Experiments

GET/experiments

Returns all experiments.

GET/experiments/:id

Returns a specific experiment by ID.

All Data

GET/all

Returns all data from all categories.

Response Format

All responses are in JSON format. Here's an example response for /api/notes:

[
  {
    "id": 1,
    "content": "Plusieurs études ont montré qu'une exposition régulière aux espaces verts est associée à une réduction du stress et à une amélioration des fonctions cognitives chez les adultes."
  },
  {
    "id": 2,
    "content": "Dans une expérience contrôlée, les participants ayant marché 20 minutes dans un parc ont présenté une baisse de 15 % du taux de cortisol, comparé à ceux ayant marché en milieu urbain."
  }
]

Google Docs Add-on

This API is designed to work with the Marvelab Google Docs Add-on. The add-on allows researchers to:

  • Insert notes, interpretations, and resources directly into Google Docs
  • Generate text using AI based on research data
  • Access methodologies and experiment details
